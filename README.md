# SnapTech - Infrastructură & Securitate Hardware

SnapTech este o platformă web dezvoltată pentru a oferi servicii premium de infrastructură (Bare-Metal, DBaaS) și audit de securitate hardware. Proiectul reprezintă combinarea exactă dintre puterea de calcul de înaltă performanță și securitatea cibernetică absolută la nivel de tranzistor.

## Funcționalități Principale

* **Design Minimalist & Bento Grid:** O interfață de utilizator (UI) modernă, cu animații fluide, asimetrie calculată matematic, creată fără framework-uri externe (Vanilla HTML/CSS/JS).
* **Knowledge Hub & Filtrare Dinamică:** Sistem integrat pentru sortarea resurselor tehnice și a echipamentelor.
* **Sistem de Autentificare (Full-Stack):** Flux complet de Înregistrare/Logare conectat direct la baza de date.
* **Gestiunea Programărilor:** Preluarea datelor clienților și stocarea lor securizată pe server, incluzând și notificarea lor în legătură cu programarea pentru a ne contacta.

## Tehnologii Utilizate

**Front-end:**
* HTML5 (Structură semantică)
* CSS3 (Flexbox, CSS Grid, efecte de Glassmorphism și Parallax)
* JavaScript (Vanilla JS pentru logica de filtrare și animații scroll)

**Back-end & Bază de Date:**
* Node.js (Serverul aplicației - `server.js`)
* MySQL (Gestiunea tabelelor `utilizatori` și `programari`)
* Integrare API și comunicare asincronă cu front-end-ul (Fetch API)
